# ac-c4-todolist 專案

# 目前已完成項目

- 以成功連結資料庫，並由 SEEDER 導入基本的 8 個餐廳資料(原 json 內容)
- 成功導入 CRUD 功能。
- 路由+flash 訊息設置與重構
- 環境變數設置
- 分頁功能

# 在本地開啟專案前的環境變數，需先在 cmd 設置以下步驟

- set NODE_ENV=development
- 建立.env 檔，並設置 SESSION_SECRET="自行輸入"
- set SESSION_SECRET="自行輸入"
- npm run dev
